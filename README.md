# budgeteer

Application for users to keep track of their annual budget. Users can record and manage their month-to-month expenses in their respective accounts. They get an overview of the data over the entire year as well.

## Tech Stack

- [Create React App](https://github.com/facebook/create-react-app)
- [Redux](https://redux.js.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- [Express](https://expressjs.com/)
- [Google Firebase](https://console.firebase.google.com)
